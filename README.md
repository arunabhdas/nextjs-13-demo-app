# nextjs-13-demo-app
Next JS 13 Demo App

## Steps

* Bootstrap nextjs-13 app as below
```
npx create-next-app@latest --experimental-app

Creating a new Next.js app in ./nextjs-13-demo-app/nextjs-13-demo-app.

Using npm.

Installing dependencies:
- react
- react-dom
- next
```

* Dev 
```
npm run dev
```
